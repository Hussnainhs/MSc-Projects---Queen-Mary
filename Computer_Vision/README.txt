Ths folder contains the implementation in MATLAB of the cmputer vision problem described in the file "computer_vision_problems.pdf".
Each of the 6 exercises has its own folder. 
Follow the following steps to run the code: 

A template Matlab file named " ICV_template_ex.. " is being created for each exercise which contains all the code necessary to run each part of the question. 
For example, in order to run the first question:
	- Open folder "ex1"
	- Open file "ICV_template_ex1.m"
	- Uncomment the section of the part of the exercise that you want to run;
	- Run the file
